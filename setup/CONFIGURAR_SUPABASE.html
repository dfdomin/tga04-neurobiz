Configurar Supabase

Activa sincronización cloud para progreso, dashboards y notas de TGA04.

1. Crear proyecto

  1. Entra a supabase.com.
  2. Crea un proyecto nuevo.
  3. Abre Project Settings → API.
  4. Copia Project URL y la clave anon / publishable.

2. Editar configuracion

Abre js/supabase-config.js y reemplaza estos valores:

var SUPABASE_URL = "https://TU-PROYECTO.supabase.co";
var SUPABASE_KEY = "TU_CLAVE_PUBLICA_ANON";

La clave anon es publica por diseño, pero las politicas RLS siguen siendo obligatorias.

3. Ejecutar SQL

  1. En Supabase abre SQL Editor.
  2. Copia el contenido de setup/supabase_schema.sql.
  3. Ejecuta el script una sola vez.
Abrir schema SQL

4. Probar flujo

  1. Abre una semana, por ejemplo semana1/index.html.
  2. Completa el modal de bienvenida.
  3. Haz clic en Guardar.
  4. Abre dashboard/profesor.html, PIN tga04, y verifica el registro.
Privacidad: este proyecto usa una arquitectura estatica para clase. Evita publicar reportes con cedulas completas fuera del entorno academico y usa la base solo para datos necesarios del curso.
