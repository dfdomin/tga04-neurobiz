📚 Marco teórico

Cuerpo al menos una vez 🟢 VIGENTE

📖
¿Qué significan las siglas HH-FHH?
HH = "Haz Hasta" (inicia el ciclo — el cuerpo se ejecuta sin preguntar primero)  ·  FHH = "Fin Haz Hasta" y aquí va la condición de parada
La condición se evalúa después del cuerpo: si es falsa desde el inicio, el ciclo igual corre una vez.
🐍 En Python: no existe do-while nativo, se usa while True: ... if condición: break — el break hace las veces de FHH.
🟡 EVOLUCIONADO La sintaxis cambió; el patrón "intenta primero, pregunta después" sigue vigente en validaciones y menús.

HH-FHH se usa cuando la interacción debe suceder antes de evaluar la condición. Por eso es ideal para formularios, validaciones y menús.

⏰
Analogía vida real: una alarma suena y luego pregunta si quieres posponerla.
🏢
Analogía empresarial: un formulario de préstamo se muestra al menos una vez antes de verificar si el cliente firmó.

☁️ Python en Google Colab — Semana 7

HH-FHH no tiene equivalente directo en Python, pero el patrón while True: ... break logra exactamente lo mismo. Abre Google Colab.

🐍
while True: crea un ciclo que no tiene condición de entrada — siempre empieza. El break al final hace las veces de FHH: cuando la condición se cumple, el ciclo se detiene.
🐍 Python — Validar cédula del paciente
while True:
    cedula = int(input("Ingrese número de cédula: "))
    if cedula > 0:
        print(f"Cédula {cedula} registrada correctamente.")
        break
    else:
        print("Error: la cédula debe ser un número positivo. Intente de nuevo.")
🐍 Python — Menú de atención NeuroBiz
while True:
    print("\n=== NeuroBiz — Menú de Atención ===")
    print("1. Nueva sesión")
    print("2. Consultar paciente")
    print("3. Ver reportes")
    print("0. Salir")
    opcion = int(input("Seleccione una opción: "))

    if opcion == 1:
        print("→ Registrando nueva sesión...")
    elif opcion == 2:
        print("→ Consultando paciente...")
    elif opcion == 3:
        print("→ Generando reportes...")
    elif opcion == 0:
        print("Hasta luego.")
        break
    else:
        print("Opción inválida. Intente de nuevo.")
