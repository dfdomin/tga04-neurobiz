🎓 Estrategias Didácticas Recomendadas

✅

Trabaje por capas: primero funciones, luego datos, luego reporte. Evita que el código completo abrume.

🧠

Concepto clave: semana 14 sintetiza todo el cuatrimestre. Nombre explícitamente cada estructura ya dominada.

⚠️

Error frecuente: confundir lista con diccionario. Use la metáfora: “lista = caja de fichas; diccionario = una ficha”.

🏛️

Mensaje de valor: crear un sistema pequeño pero completo es evidencia poderosa de transferencia del aprendizaje.
