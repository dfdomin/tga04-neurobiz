Condición de entrada 🟢 VIGENTE

📖
¿Qué significan las siglas MQ-FMQ?
MQ = "Mientras Que" (inicia el ciclo — la condición va aquí)  ·  FMQ = "Fin Mientras Que" (cierra el ciclo)
La condición se evalúa antes de entrar: si es falsa desde el inicio, el cuerpo no se ejecuta ni una vez.
🐍 En Python: while condición: — sin FMQ; Python delimita el bloque con indentación.

En MQ-FMQ la condición se prueba antes del cuerpo. Si es falsa desde el inicio, el ciclo puede ejecutarse cero veces.

Centinela, contador y acumulador

ElementoFunciónEjemplo
CentinelaMarca fin de datos-1
ContadorCuenta iteracionesregistros ← registros + 1
AcumuladorSuma resultadostotal ← total + sesiones
⚠️
Si la condición nunca se vuelve falsa, aparece un ciclo infinito. Siempre identifica qué variable cambia y cómo se alcanza la parada.

📝 MQ-FMQ con centinela

    total_acumulado: ENTERO
    registros: ENTERO

    registros ← 0

      registros ← registros + 1

☁️ Python en Google Colab — Semana 6

Traducimos MQ-FMQ a Python. Abre Google Colab y escribe este algoritmo real.

🐍
El pseudocódigo MQ condición FMQ se convierte en while condición: en Python. El cuerpo va indentado 4 espacios. No necesitas FMQ — Python sabe dónde termina por la indentación.
🐍 Python — Procesar sesiones con centinela
total = 0
contador = 0
sesion = int(input("Ingresa tasa de sesión (%) o -1 para terminar: "))

while sesion != -1:
    total += sesion
    contador += 1
    sesion = int(input("Ingresa tasa de sesión (%) o -1 para terminar: "))

if contador > 0:
    promedio = total / contador
    print(f"Sesiones procesadas: {contador}")
    print(f"Promedio de tasa: {promedio:.1f}%")
else:
    print("No se procesaron sesiones.")
⚠️
Ciclo infinito: Si olvidás actualizar la variable dentro del while (pedir la siguiente sesión), el programa nunca termina. Presiona el botón ⏹ de Colab para detenerlo.
