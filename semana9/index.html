📚 Marco teórico

Combinar estructuras 🟢 VIGENTE

Los problemas de negocio no llegan por capítulos: exigen entrada, iteración, decisión, acumulación y salida en un mismo flujo.

📑
Analogía: un informe de consultoría transforma datos crudos en análisis y luego en resumen ejecutivo.

Buenas prácticas

Nombres claros, separación entre inicialización y proceso, y comentarios breves si aclaran el propósito.

De requisitos a pseudocódigo y diagrama

Primero se comprende el requisito, luego se diseña el pseudocódigo y finalmente se representa el flujo.

Modularización 🟢 VIGENTE

Dividir el problema en subalgoritmos: leer sesión, clasificar color, acumular totales y mostrar resumen.

⚠️
Error frecuente: calcular el promedio dentro del ciclo. Debe hacerse después del acumulado completo.

📝 SistemaAnalisisNeuroBiz
ALGORITMO SistemaAnalisisNeuroBiz
  VARIABLES
    num_sesiones: ENTERO
    i: ENTERO
    tasa: REAL
    tipo: CADENA
    total_verde: ENTERO
    total_amarillo: ENTERO
    total_rojo: ENTERO
    promedio_tasa: REAL
    acumulado_tasa: REAL
  INICIO
    total_verde ← 0
    total_amarillo ← 0
    total_rojo ← 0
    acumulado_tasa ← 0
    LEER num_sesiones
    PARA i ← 1 HASTA num_sesiones FPA
      ESCRIBIR "Sesión ", i, " de ", num_sesiones
      LEER tasa
      LEER tipo
      acumulado_tasa ← acumulado_tasa + tasa
      DD tasa
        CASO >= 80.0:
          total_verde ← total_verde + 1
          ESCRIBIR "  → 🟢 VERDE"
        CASO >= 50.0:
          total_amarillo ← total_amarillo + 1
          ESCRIBIR "  → 🟡 AMARILLO"
        CASO OTRO:
          total_rojo ← total_rojo + 1
          ESCRIBIR "  → 🔴 ROJO"
      FDD
    FPA
    promedio_tasa ← acumulado_tasa / num_sesiones
    ESCRIBIR "=== RESUMEN EJECUTIVO NEUROBIZ ==="
    ESCRIBIR "Sesiones analizadas: ", num_sesiones
    ESCRIBIR "Promedio tasa retención: ", promedio_tasa, "%"
    ESCRIBIR "Estado VERDE: ", total_verde
    ESCRIBIR "Estado AMARILLO: ", total_amarillo
    ESCRIBIR "Estado ROJO: ", total_rojo
  FIN

Combinas if/elif/else + for range() + acumuladores. Es exactamente el mismo algoritmo del pseudocódigo, en lenguaje Python.
🐍 Python — Sistema de Análisis NeuroBiz (INTEGRADOR)
n = int(input("¿Cuántas sesiones analizar? ")
total_verde = total_amarillo = total_rojo = 0
acumulado_tasa = 0.0

for i in range(1, n + 1):
    print(f"\n--- Sesión {i} ---")
    tasa = float(input("  Tasa de asistencia (%): "))
    acumulado_tasa += tasa

    if tasa >= 80:
        total_verde += 1
        print("  → Verde ✅")
    elif tasa >= 60:
        total_amarillo += 1
        print("  → Amarillo ⚠️")
    else:
        total_rojo += 1
        print("  → Rojo 🔴")

print(f"\n{'='*40}")
print("RESUMEN EJECUTIVO NeuroBiz")
print(f"  Sesiones analizadas: {n}")
print(f"  Verde  (≥80%): {total_verde}")
print(f"  Amarillo(≥60%): {total_amarillo}")
print(f"  Rojo   (<60%): {total_rojo}")
if n > 0:
    print(f"  Promedio tasa: {acumulado_tasa/n:.1f}%")
