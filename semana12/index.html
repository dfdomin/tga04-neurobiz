Primer notebook NeuroBiz

🐍 Python — tasa_retención en tres celdas
# Semana 12 — NeuroBiz: Calcular tasa de retención
# Celda 1: Variables y entrada
sesiones_asistidas = int(input("Sesiones asistidas: "))
sesiones_fallidas = int(input("Sesiones fallidas: "))
nombre_terapeuta = input("Nombre del terapeuta: ")

# Celda 2: Cálculos
total = sesiones_asistidas + sesiones_fallidas
tasa_retencion = (sesiones_asistidas / total) * 100
es_activo = sesiones_asistidas > 0

# Celda 3: Resultados
print(f"=== REPORTE NEUROBIZ ===")
print(f"Terapeuta: {nombre_terapeuta}")
print(f"Sesiones totales: {total}")
print(f"Tasa de retención: {tasa_retencion:.1f}%")
print(f"Estado activo: {es_activo}")

⚡ Evaluación rápida NeuroBiz

1. En Python, ← (asignación de Cairó) se escribe como:

  1. =
  2. ←
  3. :=
  4. ==
✅ a — Python usa = para asignar valores a una variable.

2. Para mostrar "Hola NeuroBiz" en Python se usa:

  1. print("Hola NeuroBiz")
  2. ESCRIBIR
  3. console.log
  4. System.out.println
✅ a — La función estándar de salida en Python es print().

3. f"Tasa: {tasa:.1f}%" muestra la tasa con:

  1. 1 decimal
  2. 0 decimales
  3. 2 decimales
  4. En porcentaje automático
✅ a — :.1f indica formato flotante con un decimal.

4. int(input("...")) convierte la entrada a:

  1. Número entero
  2. Texto
  3. Decimal
  4. Booleano
✅ a — input() trae texto y int() lo transforma en entero.

5. Google Colab es preferido porque:

  1. No requiere instalación y funciona en el navegador
  2. Es el IDE oficial de Python
  3. Solo funciona en Windows
  4. Requiere instalar Anaconda
✅ a — Colab corre en la nube y permite empezar de inmediato.
