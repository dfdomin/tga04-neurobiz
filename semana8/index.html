📖
¿Qué significan las siglas PARA-FPARA?
PARA = "Para" (inicia el ciclo contador con variable, inicio, fin y paso)  ·  FPA = "Fin Para" (cierra el ciclo)
La variable se incrementa automáticamente en cada vuelta según el PASO (positivo o negativo).
🐍 En Python: for mes in range(inicio, fin+1, paso): — sin FPA; Python usa indentación.
El +1 en Python es necesario porque range() excluye el límite superior, diferencia importante al traducir algoritmos.

PARA-FPARA se usa cuando ya conocemos cuántas veces debe repetirse el proceso. Ejemplos: 12 meses, 52 semanas, 85 empleados.

📆
Analogía vida real: recorrer los 12 meses del año.
🏢
Analogía empresarial: nómina mensual de 85 empleados.

Sintaxis

PARA variable ← inicio HASTA fin [CON PASO incremento] FPA. El paso puede ser positivo o negativo.

Comparación de estructuras

EstructuraCuándo usarEjemplo NeuroBiz
MQ-FMQIteraciones desconocidasProcesar sesiones hasta que se acaben
HH-FHHInteracción al menos una vezValidar cédula
PARA-FPARAIteraciones exactasReporte de 12 meses
🧠
PARA sigue vigente en ETL, auditorías por lote y reportes programados.

📝 Pseudocódigo y equivalencias

📝 PARA-FPARA — Reporte mensual
ALGORITMO ReporteMensualNeuroBiz
  VARIABLES
    mes: ENTERO
    sesiones_mes: REAL
    total_anual: REAL
    promedio: REAL
  INICIO
    total_anual ← 0
    PARA mes ← 1 HASTA 12 FPA
      ESCRIBIR "Ingrese sesiones del mes ", mes, ":"
      LEER sesiones_mes
      total_anual ← total_anual + sesiones_mes
    FPA
    promedio ← total_anual / 12
    ESCRIBIR "Total sesiones 2026: ", total_anual
    ESCRIBIR "Promedio mensual: ", promedio
  FIN
📝 PARA-FPARA con PASO
ALGORITMO TablaRendimientoTerapeutas
  VARIABLES
    nivel: ENTERO
    meta: REAL
  INICIO
    ESCRIBIR "=== TABLA DE METAS POR NIVEL ==="
    PARA nivel ← 10 HASTA 100 CON PASO 10 FPA
      meta ← nivel * 8.5
      ESCRIBIR "Nivel ", nivel, "% → Meta: ", meta, " sesiones/mes"

🐍 Python — Reporte mensual NeuroBiz 2026
total_anual = 0

for mes in range(1, 13):
    ingresos = float(input(f"Ingresos mes {mes}: $"))
    total_anual += ingresos
    print(f"  Mes {mes:02d}: ${ingresos:>12,.0f}")

promedio = total_anual / 12
print(f"\n{'─'*35}")
print(f"Total anual:     ${total_anual:>12,.0f}")
print(f"Promedio mensual:${promedio:>12,.0f}")
